Ali's Vocabulary Learning Platform

Ali's Flash Card 2.0 rethinks vocabulary learning.

PYTHON (Streamlit) SQL (SQLite) LLMs (Google's Gemini)

01 / THE PROBLEM

Flashcards encourage passive learning through contextless memorization. This reinforces L1 interference by relying on translations instead of active usage.

02 / THE ACTION

I developed a local web app (Flash Denken) based on a better way to learn vocabulary. This learning platform provides contextualized and active learning/recall experience.

03 / THE RESULT

My app was a huge part of how I reached a B2 level in Dutch only a year after moving here.

Inspiration & Context

As a computational hydrologist, I often find that the most elegant solutions are found at the intersection of physical topology and digital logic. This project was inspired by the need for resilience in urban infrastructure.

Technical Implementation

Writing clean, maintainable Python was essential for ensuring the startup team could iterate on the model after the initial hand-off.
